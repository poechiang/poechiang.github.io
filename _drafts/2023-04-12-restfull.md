## Rest 动词

### GET

```java
@GetMapping("/users")
public List<User> selectUsers(){
  return userService.findUsers();
}

@GetMapping("/users/{id}")
public User findUser(@PathVariable("id") String id){
  return userService.findUserById(id);
}
```

### POST

```java
@PostMapping("/users")
@ResponseBody
public int addUser(User user){
  return userService.addUser(user);
}
```
